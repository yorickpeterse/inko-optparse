# inko-optparse

An opinionated and lightweight command-line argument and option parser for
[Inko](https://inko-lang.org).

# Requirements

- Inko `main` (for now)

# Installation

```bash
inko pkg add github.com/yorickpeterse/inko-optparse 0.1.0
inko pkg sync
```
